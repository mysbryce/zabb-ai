# Zabb AI (แซ่บเอไอ)

**Premium Local Roleplay Engine.**

- Local-only: No accounts. Data stays in your browser.
- AI Powered: High-fidelity storytelling via Groq SDK.
- Minimalist UI: Modern, animated, and immersive.
- Unfiltered: Freedom for creators.

### Setup

1. `bun install`
2. Configure AI providers and keys via the Settings Modal in the UI.
   - Or manually edit `ai-config.json` in the root directory.
3. `bun run dev`


_This is an AI-driven project._
